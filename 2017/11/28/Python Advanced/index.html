Python Advanced

函数

元组类型可变长参数

在参数名前加
`def f(list): pass`f(1,2,3,4,5)

字典类型可变长参数

在参数名前加
`def f(dict): pass`f(x=1,y=2,z=3)

高阶函数

高阶函数，是参数为函数对象的函数
函数名的赋值就是函数对象赋值
def f(x): pass
g = f
def F(f,x): f(x) 形参f的类型是函数对象
F(g,10)

函数变量作用域

Local > Enclosing > Global > Build-in

在函数内部修改global变量时，使用global关键字声明对外部变量的改写操作

匿名函数

适用于只用一次的函数，不许考虑命名
函数对象 = lambda 形参列表：表达式
例如：(通过测试)
(lambda x,y,z: x if x>y and x>y else (y if y>z else z))(3,4,5)

生成器 generator

生成器是带有无限循环的yield的函数
按照需求依次生成数据项，需要的时候才生成，节约空间
用yield生成，用__next__()获取
每次调用__next__()时，从上一次的yield语句处继续执行，直到遇到下一个yield

装饰器 decorator

包装函数的函数，在不必改动原有函数的基础上增加功能
对功能没影响、方便阅读的语法叫语法糖

闭包

一个函数把内部定义的函数作为返回值，那么这个内部定义的函数成为闭包；对enclosing作用域进行引用的内部函数也叫闭包。
闭包将函数语句和执行环境打包在一起

outer接受的参数是一种“执行环境”，经过一次初始化之后就不会改变
inner接受的参数是普通变量，可以与特定“执行环境”进行交互
装饰器就是闭包最好的例子

##面向对象

###　类与对象
用class声明一个类，括号内是继承的对象
class subclass(class):

三重引号"""括起来的是类的说明文档，可用.__doc__方法查看

构造函数__init__的第一个参数必须为self，即该类的一个实例本身
在构造函数中用self.attribute定义成员变量
用双下划线开头的标识符用于定义类的私有成员，外界不能直接访问
例如__str__(self)定义一个类的字符串返回形式
用dir(class_name)和help(class_name)可以查看类和对象中的参数情况
类的方法的第一个参数为self

库和包

Python程序的模块化架构：一个顶层程序包含主要控制流程，是程序运行时启动的文件，多个模块文件可以互相调用，可以调用库模块。
模块是Python的最高级别组织单元

Python程序的三层组织架构：包、模块、函数与类
一个包可以包含多个模块，一个模块可以有多个函数与类
包对应文件夹，类似一个工具箱；模块对应一个.py文件

import一个包（文件夹）需要在在文件夹中有一个__init__.py文件

四种包管理工具

distutils, setuptools, easy_install, pip

####包的格式
egg格式以.egg作为扩展名，用setuptools解释
ZIP格式以.whl作为扩展名

安装方法

第三方库如果提供

  • 可执行文件.exe，直接运行
  • 带setup.py文件的安装包，解压后在对应目录执行Python setup.py install命令
  • .whl压缩包，在Python\Scripts下执行命令pip.exe XXX.whl
  • .egg压缩包，在Python\Scripts下执行命令easy_install.exe XXX.egg
