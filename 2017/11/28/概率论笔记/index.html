概率

概率的公理化定义：
样本空间的子集构成事件域，对于任意事件 $A$，定义在事件域上的实值函数$P(A)$满足：

  1. 非负性（概率不为负）
  2. 正则性（全空间概率为1）
  3. 可列可加性（互斥事件的和的概率，等于概率的和）

古典概率

  1. 随机现象包含有限样本点
  2. 样本点的出现等可能性
  3. 某事件的概率等于该事件所含样本点数/所有样本点数

例子： 放回抽样、不放回抽样、盒子模型、生日问题

确定概率的几何方法
例子：会面问题、浦丰投针

确定概率的主观方法

概率的性质

  1. 不可能事件概率为0.
    但是，概率为零的事件不一定是不可能事件。

  2. 有限可加性：有限互斥事件之和的概率，等于概率的和

  3. $P(A) = 1 - P(\bar A)$
  4. 若$A\subset B$，则$P(A-B)=P(A)-P(B)$
  5. 单调性。若$A\supset B$，则$P(A) \geq P(B)$
    逆命题不成立
  6. 任意事件A,B，$P(A-B)=P(A)-P(AB)$
  7. 加法公式。任意事件,$P(A+B) = P(A) + P(B) -P(AB)$

例子：配对问题

条件概率

若$P(B) > 0$，则称$P(A|B)=\frac{P(AB)}{P(B)}$为在B发生下A的条件概率。
依然满足概率的公理化定义

  1. 乘法公式：积事件拆开条件概率
    例子： 罐子模型（波利亚模型）
  2. 全概率公式$$P(A)=\sum_iP(B_i)P(A|B_i)$$
    例子：中奖模型
  3. 贝叶斯公式$$P(B_i|A)=\frac{P(B_i)P(A|B_i)}{\sum_jP(B_j)P(A|B_j)}$$
  • 因果
  • 先验、后验
    例子：狼来了

事件独立性

  1. 一个事件是否影响另一事件
    独立性定义$$P(AB)=P(A)P(B)$$
    与事件A、B的关系无关

  2. 独立事件的对立事件也独立
    两两独立、相互独立

  3. 试验独立性
    独立试验：试验结果是独立事件
    n重独立重复试验、n重伯努利试验

随机变量

  • 随机变量：样本空间的一个实值函数，服从一定分布
  • 分布函数：$F(x) = P(X\leq x)$
    性质：
  1. 单调性：分布函数是实轴上的单调非减函数
  2. 有界性
  3. 右连续性
    这三个性质也是判别条件
  • 分布列
    性质： 非负性、正则性
  • 密度函数$$p(x) = F’(x)\F(x)=\int_\inf^{x}p(t)dt$$
    $p(x)$非负可积。
    性质：非负性、正则性

数学期望

  1. 如果无穷级数绝对收敛，那么$$E(X)=\sum_{i=1}^{inf}x_ip(x_i)$$
    称为X的数学期望
    类似可定义连续随机变量的数学期望
    数学期望是消除随机性的主要手段。
  2. 性质
  • 随机变量的函数的数学期望，把X换成g(x)即可。证明略。
  • 数学期望的线性

方差

$$Var(X)=E(X-E(X))\=\sum_i(x_i - E(X))^2p(x_i)$$
标准差与数学期望和随机变量有相同的量纲，所以常用。
随机变量的期望存在，方差不一定存在；方差存在，期望一定存在。
性质：

  1. $Var(X) = E(X^2) - [E(X)]^2$
  2. 常数方差为0
  3. $Var(aX+b) = a^2Var(X)$

切比雪夫不等式
设随机变量的期望和方差都存在，那么对于任意常数$\epsilon>0$有
$$P(|X - E(X)|<\epsilon)\geq 1 - \frac{Var(X)}{\epsilon^2}$$
或者
$$P(|X - E(X)|\geq\epsilon)\leq \frac{Var(X)}{\epsilon^2}$$
大偏差事件发生的概率的上界与方差成正比。

性质2的逆命题成立
$Var(X)=0$则$P(X=a)=1$
通过切比雪夫不等式证明

离散分布

  1. 二项分布$X$~$b(n,p)$
    $$P(X=k)=C_n^kp^k(1-p)^{n-k} \space\space\space\space k=0,1,\dots,n$$
    n重伯努利试验中p对应事件发生的次数。
  • $E(X) = np$
  • $Var(X) = np(1-p)$

  1. 两点分布：n=1的二项分布
    n个独立同为两点分布的随机变量之和服从二项分布。

  2. 泊松分布$X$~$P(\lambda)$, $\lambda > 0$
    $$P(X=k)=\frac{\lambda^k}{k!}e^{-\lambda} \space\space\space k=0,1\dots$$
    单位时间上的发生次数。

  • $E(X) = \lambda$
  • $Var(X) = \lambda$
  • 泊松定理： 用来估计n很大、p很小的二项分布。
  1. 超几何分布$X$~$h(n,N,M)$
    $$P(X=k)=\frac{CM^kC{N-M}^{n-k}}{C_N^n}, k=0,1,\dots,min{n, M}$$
    从含有M件不合格产品的N件产品中不放回抽样n件，得到的不合格产品数。
  • $E(X) = nM/N$
  • 当n$<<$N时，不放回抽样近似为放回抽样，超几何分布可以用二项分布近似
  1. 几何分布$X$~$Ge(p)$
    $$P(X=k)=(1-p)^{k-1}p\space\space\space k=1,2\dots$$
    在伯努利试验序列中，p对应事件首次出现时的试验次数。
  • $E(X) = \frac{1}{p}$
  • $Var(X) = \frac{1-p}{p^2}$
  • 无记忆性：$P(X>m+n|X>m) = P(X>n)$
  1. 负二项分布
    $$P(X=k)=C_{k-1}^{r-1}p^r(1-p)^{k-r} \space\space\space k=r,r+1,r+2\dots$$
    伯努利试验序列中，p对应的事件第r次出现时的试验次数。
  • 均值$n/p$
  • 服从负二项分布的随机变量可以表示成n个独立同几何分布的随机变量之和

连续分布

  1. 正态分布$X$~$N(\mu, \sigma^2)$
    $$P(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}} \space, -\infty < x < +\infty$$
  • 钟形曲线、对称性、拐点
  • 均值和方差对曲线形状的影响
  • 分布函数的图像：S型、类似sigmoid
  • 标准正态分布$N(0, 1)$的密度函数和分布函数
  • 任意正态分布都可以通过线性变换化为标准正态分布。
  • 3$\sigma$原则
  1. 均匀分布$X$~$U(a,b)$
    $$P(x)=\begin{cases} \frac{1}{b-a} &\text{a<x<b} \
    0 & \text{else}
    \end{cases}$$
  • 密度函数有突变，分布函数有折点
  • 均值${a+b}\over {2}$
  • 方差${(b-a)^2\over 12}$
  1. 指数分布$X$~$Exp(\lambda)\space \lambda > 0$
    $$P(x)=\begin{cases}\lambda e^{-\lambda x} &,\text{x $\geq$ 0}\ 0 &,\text{x < 0}
    \end{cases}$$
    器件使用寿命的分布
  • 分布函数形式：从1开始负指数衰减
  • 均值$1/\lambda$
  • 方差$1/\lambda^2$
  • 无记忆性: $P(X>m+n|X>m) = P(X>n)$
  • 与泊松分布的关联
    若在一定时间内，事件发生次数服从参数为$\lambda t$的泊松分布，那么相继两次事件之间的时间间隔服从参数为$\lambda$的指数分布
  1. Gamma分布

  • Gamma函数 $G(\alpha) = \int_0^\infty x^{\alpha -1}e^{-x}dx$
    参数$\alpha > 0$
    性质：

    1. $G(1) =1 $
    2. $G(\frac{1}{2})=\sqrt{\pi}$
    3. $G(\alpha + 1) = \alpha G(\alpha)$
    • Gamma分布$X$~$Ga(\alpha, \lambda)$
      $$p(x)=\begin{cases} \frac{\lambda^\alpha}{G(\alpha)}x^{\alpha -1}e^{-\lambda x} &\text{x $\geq$ 0}\
      0 &\text{x < 0}\end{cases}$$
  • 期望$\frac{\alpha}{\lambda}$
  • 方差$\frac{\alpha}{\lambda^2}$
  • $Ga(1, \lambda) = Exp(\lambda)$
  • $Ga(\frac{n}{2}, \frac{1}{2}) = \chi^2(n)$ 自由度为n的卡方分布
  1. Beta分布
  • Beta函数$$B(a,b)=\int_0^1 x^{a-1}(1-x)^{b-1}dx$$
    参数$a>0, b>0$
    性质：
  1. $B(a,b) = B(b,a)$
  2. $B(a,b) = \frac{G(a)G(b)}{G(a+b)} $ (神换元)
  • Beta分布$X$~$Be(a,b)$, $a>0, b>0$
    $$p(x) = \begin{cases} \frac{G(a)G(b)}{G(a+b)}x^{a-1}(1-x)^{b-1} &\text{0<x<1}\ 0 &\text{else} \end{cases}$$
  • 期望$\frac{a}{a+b}$
  • 方差$\frac{ab}{(a+b)^2(a+b+1)}$

##随机变量函数的分布
随机变量的函数也是随机变量

  1. 离散：易求
  2. 连续
    当$g(x)$单调时，
    $$p(g(x)) = \begin{cases} p(g^{-1}(y))|\frac{d g^{-1}(y)}{dy}| &\text{y$\in$ g(x)值域}\
    0 &\text{else}\end{cases}$$
    可以证明：
  • 正态分布变量的线性函数仍是正态分布，均值做线性改变，方差做平方改变
  • 关于正态分布变量的e指数函数，是对数正态分布.
  • 若$X$~$Ga(\alpha, \lambda)$, k > 0，则$kX$~$Ga(\alpha, \lambda / k)$。任意Gamma分布可以转化为开方分布。
  • 把任意随机变量代到它的分布函数中，得到的随机变量服从(0,1)均匀分布。所以用均匀分布可以产生任意一个分布。

##其他特征数

  1. k阶矩
  • k阶原点矩$E(X^k)$
  • k阶中心矩$E(X-E(X))^k$
  1. 变异系数
    $G_v(X) = \frac{\sqrt{Var(X)}}{E(X)}$
    是一个无量纲量，消除了量纲对波动情况的影响。
  2. p分位数
    $F(xp)=\int{-\infty}^{x_p} p(x)dx = p$
  3. 中位数： p=0.5
  4. 偏度系数
    $\beta_s = \frac{E(X-E(X))^3}{Var(X)^{3/2}}$
    衡量对称性的偏离
    小于0左偏，大于0右偏
  5. 峰度系数
    $\beta_k = \frac{E(X-E(X))^4}{Var^2(X)}$
    以正态分布为基准，衡量尖峭程度或尾部粗细
